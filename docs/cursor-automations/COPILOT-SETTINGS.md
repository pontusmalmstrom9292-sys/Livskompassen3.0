# GitHub Copilot Pro — repo-inställningar (manuellt)

Gör detta en gång i GitHub (Settings → Copilot):

1. **Code review:** Auto-review på PR du skapar
2. **Copilot Memory:** På (Pro)
3. **Dependabot:** Stäng/ignorera major bumps utan PMIR (Tailwind 4, TS 6, firebase-functions 7)

Repository instructions läses från `.github/copilot-instructions.md` och `.github/instructions/*.instructions.md`.
