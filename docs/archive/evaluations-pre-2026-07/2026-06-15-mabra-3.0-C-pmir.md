## 1. Syfte

Denna PMIR är **implementationsgate** för minimal viable leverans av tre gröna pelare under MåBra 3.0:

| Kat | Titel | MV-mål |
|-----|-------|--------|
| **2** | Fitness / Rörelse | Deterministisk mikrorörelse-panel + `mabra_sessions` metadata |
| **3** | Näring / vätska | Skuld-fri hydrering/markering — lokal eller mutable logg |
| **6** | Prova nya saker | **Redan levererad** — M3.0-C = kapacitetskoppling + pelar-wire |

**Denna PMIR gör inte:** skriva prod-kod, deploya rules, eller kurera ny bank utan dirigent.

## 4. Kapacitetsregler (Infinite Evolution)

**Kanonisk resolver:** `resolveCapacityLevel()` i `lib/goalDetection.ts` — primärt `evolution_hub.pillars.kognitiv.level`, fallback ekonomi-nivå.

```157:168:src/modules/features/dailyLife/wellbeing/mabra/lib/goalDetection.ts
export function resolveCapacityLevel(
  evolutionDoc: EvolutionHubDoc | null | undefined,
  economyLevel: EconomyCapacityLevel,
): 1 | 2 | 3 {
  const kognitivLevel = evolutionDoc?.pillars?.kognitiv?.level;
  if (typeof kognitivLevel === 'number' && kognitivLevel >= 1 && kognitivLevel <= 3) {
    return kognitivLevel as 1 | 2 | 3;
  }
  if (economyLevel === 'critical' || economyLevel === 1) return 1;
  if (economyLevel === 2) return 2;
  return 3;
}
```

| Nivå | Kat 2 Fitness | Kat 3 Näring | Kat 6 Explore |
