# Livskompassen v2 – Fullständig CI/CD Pipeline (GitHub Actions)

```yaml
name: Livskompassen CI/CD Pipeline

on:
  push:
    branches:
      - main
  pull_request:
    branches:
      - main

jobs:
  # 1. Bygg och testa frontend
  frontend:
    runs-on: ubuntu-latest
    name: Build & Test Frontend
    defaults:
      run:
        working-directory: ./src

    steps:
      - name: Checkout code
        uses: actions/checkout@v3

      - name: Setup Node.js 20
        uses: actions/setup-node@v3
        with:
          node-version: 20

      - name: Install dependencies (frontend)
        run: npm ci

      - name: Run ESLint (frontend)
        run: npm run lint

      - name: Run unit and integration tests (frontend)
        run: npm test -- --coverage

      - name: Build frontend (Vite/PWA)
        run: npm run build

      - name: Upload frontend build artifact
        uses: actions/upload-artifact@v3
        with:
          name: frontend-build
          path: dist/

  # 2. Bygg och testa backend (Cloud Functions)
  backend:
    runs-on: ubuntu-latest
    name: Build & Test Backend
    defaults:
      run:
        working-directory: ./functions

    steps:
      - name: Checkout code
        uses: actions/checkout@v3

      - name: Setup Node.js 20
        uses: actions/setup-node@v3
        with:
          node-version: 20

      - name: Install dependencies (backend)
        run: npm ci

      - name: Run ESLint (backend)
        run: npm run lint

      - name: Run unit tests (backend)
        run: npm test -- --coverage

      - name: Build backend (TypeScript)
        run: npm run build

      - name: Upload backend build artifact
        uses: actions/upload-artifact@v3
        with:
          name: backend-build
          path: lib/

  # 3. Säkerhetsskanning med Snyk och Dependabot
  security-scan:
    runs-on: ubuntu-latest
    name: Security Scan
    steps:
      - name: Checkout code
        uses: actions/checkout@v3

      - name: Setup Node.js 20
        uses: actions/setup-node@v3
        with:
          node-version: 20

      - name: Install dependencies
        run: npm ci

      - name: Run Snyk test (requires SNYK_TOKEN secret)
        uses: snyk/actions/node@master
        env:
          SNYK_TOKEN: ${{ secrets.SNYK_TOKEN }}
        with:
          command: test

  # 4. Deployment (efter lyckade tester och skanning)
  deploy:
    runs-on: ubuntu-latest
    needs: [frontend, backend, security-scan]
    if: github.ref == 'refs/heads/main'
    name: Deploy to Firebase Hosting and Functions

    steps:
      - name: Download frontend build artifact
        uses: actions/download-artifact@v3
        with:
          name: frontend-build
          path: dist

      - name: Download backend build artifact
        uses: actions/download-artifact@v3
        with:
          name: backend-build
          path: functions/lib

      - name: Checkout code
        uses: actions/checkout@v3

      - name: Setup Firebase CLI
        uses: w9jds/firebase-action@v2.2.0
        with:
          args: --version

      - name: Deploy to Firebase
        uses: FirebaseExtended/action-hosting-deploy@v0
        with:
          repoToken: ${{ secrets.GITHUB_TOKEN }}
          firebaseServiceAccount: ${{ secrets.FIREBASE_SERVICE_ACCOUNT }}
          projectId: livskompassen-project-id
```

---

## Pipelinebeskrivning

- **Körs vid push och pull request till main-branch.**
- Steg 1 & 2: Separat bygg och test av frontend och backend (modulär och snabb).
- Steg 3: Säkerhetsskanning för sårbarhetsanalys med Snyk.
- Steg 4: Deployment till Firebase sker endast om alla tidigare steg är godkända.
- Använder GitHub secrets för tokens och credentials.
- Bygg-artefakter hanteras för effektiv pipeline.
