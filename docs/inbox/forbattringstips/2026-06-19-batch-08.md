# 1. Kodexempel

## 1.1 Frontend: Smart Inkast UploadText (React + Zustand)

```tsx
// src/modules/supermodules/SmartInkast/components/UploadText.tsx
import React, { useState } from 'react';
import { useInkastStore } from '../../hooks/useInkastState';
import { v4 as uuidv4 } from 'uuid';

const UploadText: React.FC = () => {
  const [text, setText] = useState('');
  const addUpload = useInkastStore(state => state.addUpload);

  const handleUpload = () => {
    if (!text.trim()) return alert('Fyll i texten');
    const entry = {
      id: uuidv4(),
      type: 'text' as const,
      content: text,
      createdAt: Date.now(),
      synced: false,
    };
    addUpload(entry);
    setText('');
  };

  return (
    <div className="p-4">
      <textarea
        className="border p-2 w-full rounded"
        rows={4}
        placeholder="Skriv ditt inkast här"
        value={text}
        onChange={e => setText(e.target.value)}
      />
      <button
        disabled={!text.trim()}
        onClick={handleUpload}
        className="btn-primary mt-2"
      >
        Ladda upp textinkast
      </button>
    </div>
  );
};

export default UploadText;
```

## 1.2 Backend: submitInkastLite (Cloud Function)

```typescript
// functions/src/callables/submitInkastLite.ts
import * as functions from 'firebase-functions';
import * as admin from 'firebase-admin';
import { classifyWithVertexAI } from '../ai/vertexService';

admin.initializeApp();

export const submitInkastLite = functions.https.onCall(async (data, context) => {
  if (!context.auth) throw new functions.https.HttpsError('unauthenticated', 'User not signed in.');

  const { type, content } = data;
  if (!['text', 'image', 'voice'].includes(type)) {
    throw new functions.https.HttpsError('invalid-argument', 'Invalid content type');
  }

  const docRef = admin.firestore().collection('inkastUploads').doc();
  await docRef.set({
    userId: context.auth.uid,
    type,
    content,
    createdAt: admin.firestore.FieldValue.serverTimestamp(),
    synced: false,
  });

  const classification = await classifyWithVertexAI(content);

  await docRef.update({ classification });
  return { id: docRef.id, classification };
});
```

# 2. UX-scenarion

## 2.1 Smart Inkast - Användarresa

**Scenario:** Anna vill snabbt lägga till en idé via textupload.

- Anna öppnar appens Smart Inkast modul, ser en lugn och tydlig textvy.
- Hon skriver sin anteckning och trycker "Ladda upp".
- Under offline-läge sparas texten lokalt och synkas automatiskt när nätet är tillbaka.
- AI ger en direkt feedback om vilken kategori (silo) anteckningen hör till.
- Om Anna är ny visas adaptiv onboarding med korta steg, guidade av en AI-assistent.

## 2.2 Adaptiv Onboarding - Användarresa

- Vid första besöket aktiveras React Joyride stegvis guide.
- AI analyserar Annas beteende och anpassar tips och komplexitet.
- Anna kan när som helst få hjälp via AI-chat, integrerad i onboardingfönstret.
- När onboarding är klar loggas resultat för framtida förbättringar.
