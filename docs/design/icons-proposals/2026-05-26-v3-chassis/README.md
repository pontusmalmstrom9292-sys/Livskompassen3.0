# Ikoner v3 — Helros-chassi × 5 stilar × 10 kategorier

**DNA:** samma som låsta **D1** (disk `#3d3420`→`#080808`, guldgradient, eld, glöd) + **M3** (teal skiva möjlig i framtida variant) + yttre **tick-ring** som hem-kompassen.

| # | Chassi-stil | Beskrivning |
|---|--------------|-------------|
| 1 | **Helros** | Streckad innerring + kors + nordstjärna |
| 2 | **Stjärn-eld** | Yttre polygon + eld-nåd |
| 3 | **Sacred** | 6-strålar från centrum + streckad båge |
| 4 | **Pansar** | Trippel guldring |
| 5 | **Eldnål** | Kraftig nordpil + yttre båge |

**Kategorier (glyph i mitten):** Familjen · Hamn · Valv · Dagbok · Planering · MåBra · Hero rutiner · ekonomi · utveckling · kunskap.

## Filer

50 SVG: `*/F1-helros.svg` … `Kn5-eldnal.svg`

## Regenerera

```bash
npm run icons:proposals-v3
```

## Förhandsvisa

Öppna [`preview.html`](./preview.html) i webbläsaren.

## Välja & bygga in

Per kategori välj **ett siffertecken 1–5** (chassi). Exempel: `F3, H1, V5, J2, P4, A1, R5, E3, U2, Kn4` — skriv i Cursor för kodbyte.

Äldre enkla förslag finns kvar i `2026-05-26-remaining/` men **v3** är avsedda som primära kandidater visuellt.
