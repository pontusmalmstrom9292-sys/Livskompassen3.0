# CURSOR-PROMPT FORMAT (OBLIGATORISKT)

Svara användaren kort på svenska — sedan detta block (engelska):

---
NÄSTA STEG (kopiera till Cursor):

MODEL TIER: [FAST | HEAVY]
SCOPE: [frontend-only | backend-only | rules-only | functions-only]
READ FIRST (mandatory):
  - [exact/path/to/file]
CONTEXT: [modul/zon]
LOCKED UX: [none | lista]
TASK: [Ett atomsteg. Imperativ. Namnge filer.]
MUST NOT:
  - [förbjudna åtgärder]
VERIFY (run before claiming done):
  - cd functions && npm run build   (if backend)
  - npm run build                   (if frontend)
  - npm run smoke:locked-ux         (if UX/routes)
  - npm run smoke:locked-icons        (if D1/M2/WH1/WH2)
  - npm run smoke:orkester            (if agents/synapser)
  - npm run smoke:innehall            (if content)
DONE WHEN: [exakt pass-kriterium]

Jämför dina ändringar mot hela projektets kontext. Arbeta autonomt och sluta inte förrän koden är helt felfri och appen går att använda.
---

MODEL TIER: FAST = UI/CSS/copy/enfil. HEAVY = Valv, rules, synapser, säkerhet.
