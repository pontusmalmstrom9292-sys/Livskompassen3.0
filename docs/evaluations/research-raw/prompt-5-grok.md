# Prompt 5 — Evolution utan gamification

**Modell:** Grok 4.20 Reasoning  
**Status:** ✅ Klar  
**Datum:** 2026-06-18

---

## Råsvar

### 1. Lära sig hjälpsamma modes utan tråkighet
- Innehållsbaserad anpassning (teman, inte belöningar)
- Tydlig men lågmäld feedback (belysning/skugga, ej kvantifierat)
- Interaktivitet utan krav (hoppa över, återvända)
- Variation i presentation (lista, stapel, grupp)
- Narrativt stöd (förstå syftet med varje läge)

### 2. Fem sätt att rotera kort
1. Tematisk rotation
2. Tidsbaserad (tid på dagen/vecka)
3. Slump + kontext (senaste interaktioner)
4. Fördjupningsrotation (grund → avancerat, utan krav)
5. Användarstyrd rotation

### 3. Nytt läge utan dopaminfällor
- Informativ intro, neutral ton
- Lågtröskel, ingen auto-påminnelse
- Mjuk övergång i meny, inga blinkningar
- Inga badges/nummer/brådska-färger
- «Annan vy», inte utmaning

### 4. Parallell Barnporten → vuxen
- Mognadsanpassning (livsfas, inte bara ålder)
- Flexibla zoner (välj, inte påtvingas)
- Djup/bredd-val i innehåll
- Igenkänning i design mot Barnporten
- Diskret separering utan hierarki/prestation
