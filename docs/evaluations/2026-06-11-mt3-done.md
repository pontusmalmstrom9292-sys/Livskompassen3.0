# MT-3 C1 — Flat drawer rows (MENU-DRAWER-KANON)

**Date:** 2026-06-11  
**Branch:** `main`  
**Scope:** C1 only (Drawer IA)

## Decision

Replace `DrawerHubAccordion` expandable hub sub-rows in the Vardag drawer section with **four flat rows** per [`MENU-DRAWER-KANON.md`](../design/references/MENU-DRAWER-KANON.md):

| Drawer row | Route | Notes |
|------------|-------|-------|
| Hem — Skriv | `/` | Unchanged |
| Liv och göra | `/liv` | Redirects to `LivLauncherPage` (`/vardagen`) |
| Familj och gränser | `/familj` | Redirects to `FamiljenPage` (`/familjen?tab=reflektion`) |
| Inställningar | `/installningar` | Moved from footer into Vardag list |

## Rationale

- Kanon (2026-06-01 supersidor): tabs belong **inside shell pages** (`?tab=`), not as drawer accordion children.
- Accordion sub-rows contradicted locked UX and increased cognitive load in the hamburger menu.
- Valv section unchanged: flat rows when `vaultOpen`, panic lock preserved.

## Implementation

| File | Change |
|------|--------|
| `navTruth.ts` | `vardagen.path` → `/liv`; `familjen` label → Familj och gränser, path → `/familj` |
| `NavigationDrawer.tsx` | `DRAWER_VARDAG_ITEMS.map` flat rows; removed accordion + footer settings duplicate |
| `drawerFromNavTruth.ts` | Added `isVardagDrawerRowActive` (hub-aware active state for `/liv`/`/familj` redirects) |
| `DrawerHubAccordion.tsx` | Kept for `isDrawerLinkActive` utility (Valv rows); no longer mounted in drawer |
| `smoke_locked_ux.mjs` / `smoke_superhub.mjs` | Guards for 4-row pattern; assert no `DrawerHubAccordion` in `NavigationDrawer` |
