# shared/ui

Small, prop-driven primitives. Export from `index.ts`:

- **Button** — pill variants
- **Card** — thin wrapper
- **BentoCard** — Obsidian Calm hub card (canonical)
- **Input** — form field
- **Modal** — dialog shell

Import: `import { BentoCard, Modal } from '@/shared/ui'` or `@/shared`.
