# shared/ui

Small, prop-driven primitives. Export from `index.ts`:

- **Button** — pill variants
- **Card** — thin wrapper
- **BentoCard** — Obsidian Calm hub card (canonical)
- **Input** — form field
- **Modal** — dialog shell

Import the module-local primitives directly:

`import { BentoCard, Button } from '@/shared/ui'`
